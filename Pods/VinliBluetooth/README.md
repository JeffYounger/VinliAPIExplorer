# Vinli Bluetooth SDK for iOS

A framework for communicating with a Vinli device. For more information: http://dev.vin.li

##Documentation
http://cocoadocs.org/docsets/VinliBluetooth


##Installation

Install With CocoaPods

```objective-c
pod 'VinliBluetooth'
```
